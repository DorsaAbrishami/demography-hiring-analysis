# Demography Hiring Analysis

This project analyzes demographic patterns in U.S. employment using Bureau of Labor Statistics (BLS) data.

## Data Sources
- Employment Level - Men
- Employment Level - Women
- Employment Level - Age 25–54
- Unemployment Rate - Men
- Unemployment Rate - Women
- Labor Force Participation Rate - Men
- Labor Force Participation Rate - Women

## Tools
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Goal
To analyze employment, unemployment, and labor force participation across demographic groups and identify meaningful trends and disparities.
